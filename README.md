# 📮 Postcard Maker

Transform your photos into beautiful, customizable postcards — right in your browser.

**[Live Demo →](https://gundalf.github.io/postcard/)**

## ✨ Features

- **Upload** — Drag & drop or click to upload any image
- **Text** — Add custom messages with multiple font options
- **Styling** — Adjust font size, color, and position
- **Borders** — Choose from simple, double, rounded, or vintage frames
- **Filters** — Apply B&W, sepia, warm, cool, or vintage effects
- **Orientation** — Switch between landscape and portrait
- **Export** — Download as PNG or JPG

## 🚀 Usage

1. Upload an image
2. Customize text, borders, and filters
3. Download your postcard

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript
- [html2canvas](https://html2canvas.hertzen.com/)
